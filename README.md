# SoloCards

Dragon-themed Electron app for Solo / Patience with drag-and-drop card movement.

## Run The Game

### First Start

Run these commands the first time you set up the project:

1. `npm install`
2. `npm start`

### Later Runs

After the first install, you do not need to install dependencies again.

Run:

`npm start`

This launch path works on both macOS and Windows.

## Included

- Electron desktop shell
- Klondike-style Solo / Patience layout
- Drag-and-drop tableau and foundation moves
- Stock, waste, foundations, new game, and auto-move
